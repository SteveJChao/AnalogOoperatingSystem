# AnalogOoperatingSystem
一个简单的模拟操作系统，包含常见的进程调度相关算法的实现、内存分配相关算法的实现（连续，离散）以及模拟磁盘、文件系统的实现。各模块联动，辅助辅以一系列GUI界面动画演示。
    作为操作系统课程设计，与另一名同学合作完成。
